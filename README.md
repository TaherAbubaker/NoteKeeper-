# NoteKeeper 📝

==============================
## Features 🚀
- 📝 **Create** new notes with a title and body.
- ✏️ **Edit** existing notes.
- 🗑️ **Delete** notes.
- 🔍 **Search** notes by title or content.
- 📜 **View** notes in a formatted way.
- 💾 Automatically saves your notes in `notes.json`.
